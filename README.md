# Smart Home Visit Doctor Booking Platform
A full-stack healthcare appointment booking platform that allows users to book home visit doctor appointments online with advanced healthcare features such as symptom analysis, multilingual support, and automatic prescription generation.

## Features
### Patient Features
- User Registration & Login Authentication
- Book Home Visit Appointments
- View & Manage Appointments
- Profile Management
- Search Doctors by Specialization
- Symptom Analysis System
- Multi-language Support

### Doctor Features
- Doctor Dashboard
- View Patient Appointments
- Manage Profile Information
- Generate Digital Prescriptions
- Patient Health Information Access

### Admin Features
- Admin Dashboard
- Add & Manage Doctors
- View All Appointments
- Monitor Platform Activity
- Manage Users & Doctors

## Advanced Functionalities
- AI-based Symptom Analysis
- Multi-language Support
- Automatic Prescription Generation
- Cloudinary Image Upload
- JWT Authentication & Authorization

## Tech Stack
### Frontend
* React.js
* Tailwind CSS
* Axios

### Backend
* Node.js
* Express.js

### Database
* MongoDB

### Authentication & APIs
* JWT Authentication
* Cloudinary
* Google Gemini API

## Installation Steps
1️. Clone Repository

2️. Install Dependencies
# Frontend
```bash
cd frontend
npm install
```
# Admin
```bash
cd admin
npm install
```
# Backend
```bash
cd backend
npm install
```

3. Run Project
# Frontend
```bash
npm run dev
```
# Admin
```bash
npm run dev
```
# Backend
```bash
npm start
```

## Environment Variables
Create a `.env` file inside the backend folder.

```env
MONGODB_URI=your_mongodb_url
JWT_SECRET=your_secret_key

CLOUDINARY_NAME=your_cloudinary_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_SECRET_KEY=your_secret

GEMINI_API_KEY=your_google_gemini_api_key
```
